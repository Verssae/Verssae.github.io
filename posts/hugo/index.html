Hugo-blog (README)

September 28, 2021 · 1 min · Verssae | Suggest Changes

A blog powered by HUGO

Requirements#

  • golang
  • hugo
brew install hugo

Installation#

  • Clone submodules (public, themes/PaperMod)
git submodule update --init
git submodule foreach checkout main

New Post#

hugo new posts/<title.md>

Run server#

hugo server -D

Compile#

hugo -t PaperMod

Publish#

cd public
git add *
git commit
git push origin main

Notices#

  • When you want to commit, commit submodules first
  • hugo
  • usage
